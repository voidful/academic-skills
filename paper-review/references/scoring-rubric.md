# 評分標準：完整定義

本文件詳細定義學術論文審稿的所有評分標準，包含 Overall Score、Confidence 與各維度的評分。

---

## Overall Score（整體分數）：1-10

### 10 分 — 頂級論文 (Top Paper / Strong Accept)

- 幾乎無可挑剔的優秀論文
- 在新穎性、技術深度、實驗品質上全面出色
- 可能成為該領域的經典或高引用論文
- 即使最嚴格的審稿者也難以找到重大缺陷
- **出現頻率**：極為罕見，每個會議僅有少數幾篇

### 9 分 — 優秀論文 (Excellent / Accept)

- 整體品質優異，在多個維度上表現突出
- 可能有極少數的小缺點，但不影響核心貢獻
- 明顯高於接受門檻
- 會為會議增添重要價值
- **出現頻率**：約佔接受論文的前 10-15%

### 8 分 — 強論文 (Strong / Accept)

- 品質良好，有明確的貢獻
- 可能有少數可改進之處，但優點明顯大於缺點
- 舒適地高於接受門檻
- **出現頻率**：約佔接受論文的 20-30%

### 7 分 — 好論文 (Good / Weak Accept)

- 品質不錯，有有意義的貢獻
- 有一些缺點但不致命
- 略高於接受門檻
- 可能需要在 camera-ready 中進行修改
- **出現頻率**：接受論文中的大多數落在此區間

### 6 分 — 尚可 (Marginally Above Threshold / Borderline Accept)

- 有一定的貢獻，但也有明顯的不足
- 優點與缺點大致相當，但優點略多
- 剛好在接受門檻上或略高
- 如果 rebuttal 能解決主要問題，可以接受
- **出現頻率**：borderline 論文，決定常取決於 AC 和其他 reviewer

### 5 分 — 邊界 (Borderline / Marginally Below Threshold)

- 有一些可取之處，但缺點較為明顯
- 核心想法可能有趣，但執行或呈現不夠好
- 略低於接受門檻
- 如果 rebuttal 非常有說服力，可能翻轉
- **出現頻率**：borderline 論文的另一端

### 4 分 — 低於門檻 (Below Threshold / Weak Reject)

- 有一個或多個重大問題
- 可能有有趣的想法，但執行有明顯缺陷
- 低於接受門檻
- 需要大幅修改才有可能接受
- **出現頻率**：被拒論文中較好的一端

### 3 分 — 不合格 (Not Good Enough / Reject)

- 有多個重大問題，或有一個致命缺陷
- 貢獻不足以支撐一篇完整論文
- 明顯低於接受門檻
- **出現頻率**：被拒論文中的常見分數

### 2 分 — 嚴重不足 (Poor / Strong Reject)

- 技術上有根本性問題，或品質極低
- 可能有嚴重的數學錯誤、實驗設計缺陷
- 距離接受門檻很遠
- **出現頻率**：品質明顯不足的論文

### 1 分 — 完全不合格 (Very Poor / Strong Reject)

- 不符合學術論文的基本要求
- 可能有嚴重的學術不端嫌疑
- 可能完全不在會議的範疇內
- **出現頻率**：極為罕見，通常表示有根本性的問題

---

## Confidence（信心度）：1-5

### 5 分 — 非常確信 (Very Confident)

- 我是這個具體子領域的專家
- 我非常熟悉所有相關工作
- 我有能力驗證論文中的所有技術細節
- 我對自己的評估非常有信心
- **適用場景**：審稿者就是做這個方向的、發表過多篇相關論文

### 4 分 — 確信 (Confident)

- 我熟悉這個領域的主要工作
- 我能理解論文的技術細節
- 我可能不熟悉所有最新的相關工作
- 我對自己的評估相當有信心
- **適用場景**：審稿者做的是相近方向、對領域有良好掌握

### 3 分 — 中等信心 (Moderately Confident)

- 我了解這個領域的基本概念
- 我能理解論文的主要思路，但部分細節可能不確定
- 我可能遺漏了一些相關工作
- 我的評估大致可靠，但可能有盲點
- **適用場景**：審稿者是相關但不完全相同的領域

### 2 分 — 不太確信 (Not Very Confident)

- 部分技術內容超出我的專業範圍
- 我對某些 claims 的正確性無法完全判斷
- 我可能不了解該領域的標準做法
- 我的評估可能有遺漏或偏誤
- **適用場景**：審稿者的專業與論文有一定距離

### 1 分 — 不確信 (Not Confident)

- 這基本上不是我的領域
- 我只能從很高的層面評估這篇論文
- 我的技術評估可能不可靠
- 我的意見應被賦予較低的權重
- **適用場景**：emergency review 或非常跨領域的論文

## 各維度評分標準（1-5）

以下五個維度都使用相同的 1-5 分量表。

### 通用量表

| 分數 | 等級 | 一般含義 |
|------|------|---------|
| 5 | 優秀 (Excellent) | 在該維度上幾乎無可挑剔 |
| 4 | 好 (Good) | 在該維度上表現良好，有小的改進空間 |
| 3 | 中等 (Adequate) | 在該維度上達到基本要求，有明顯的改進空間 |
| 2 | 不足 (Below Average) | 在該維度上有明顯問題，需要大幅改進 |
| 1 | 嚴重不足 (Poor) | 在該維度上有根本性問題 |

---

### 維度 1：貢獻新穎性 (Novelty & Contribution)

| 分數 | 描述 |
|------|------|
| 5 | 開創性的想法或方法；可能開啟新的研究方向；令人眼前一亮 |
| 4 | 新穎且有意義的貢獻；相對於 prior work 有清楚的進步；非 trivial 的延伸 |
| 3 | 有一定新意但不特別突出；可能是合理的延伸或組合，但有足夠的技術挑戰 |
| 2 | 新意有限；增量改進或簡單的方法組合；與已有工作的區別不大 |
| 1 | 基本沒有新意；已知結果的重述；trivial 的延伸 |

### 維度 2：寫作清晰度 (Clarity & Writing)

| 分數 | 描述 |
|------|------|
| 5 | 寫作極佳——流暢、清晰、引人入勝；非專家也能理解主要貢獻；圖表精美 |
| 4 | 寫作良好——條理清楚，易於跟隨；偶有可改進之處但不影響理解 |
| 3 | 寫作尚可——基本能理解但需要努力；有些地方不夠清楚或邏輯跳躍 |
| 2 | 寫作較差——多處難以理解；邏輯混亂或符號不一致；需要大幅重寫 |
| 1 | 寫作極差——基本無法理解論文在說什麼；嚴重的表達和邏輯問題 |

### 維度 3：實驗嚴謹性 (Experimental Rigor)

| 分數 | 描述 |
|------|------|
| 5 | 實驗設計完美——全面的 baselines、充分的統計分析、高度可重現 |
| 4 | 實驗設計良好——包含主要 baselines、有合理的分析、可重現性佳 |
| 3 | 實驗設計尚可——有基本的比較，但可能缺少一些 baselines 或統計分析 |
| 2 | 實驗設計不足——缺少重要 baselines、沒有統計分析、公平性存疑 |
| 1 | 實驗設計有根本問題——可能有 data leakage、嚴重的不公平比較 |

### 維度 4：評估完整性 (Evaluation Completeness)

| 分數 | 描述 |
|------|------|
| 5 | 評估極為全面——多資料集、完整 ablation、深入分析、局限性討論 |
| 4 | 評估良好——涵蓋主要面向，有 ablation 和分析，有一些小的遺漏 |
| 3 | 評估基本完整——有基本的實驗，但缺少 ablation 或深入分析 |
| 2 | 評估不完整——資料集太少、缺少 ablation、分析淺薄 |
| 1 | 評估嚴重不足——僅有極少量實驗，無法支撐論文的 claims |

### 維度 5：方法健全性 (Soundness of Method)

| 分數 | 描述 |
|------|------|
| 5 | 方法完全健全——理論嚴謹、假設合理、推導正確、與實驗一致 |
| 4 | 方法基本健全——技術上正確，假設大致合理，可能有小的改進空間 |
| 3 | 方法尚可——大方向正確但有一些可疑之處，假設的合理性有待商榷 |
| 2 | 方法有明顯問題——技術上有錯誤或不合理的假設，需要修正 |
| 1 | 方法有根本性錯誤——核心推導有誤、假設不成立、結論不可靠 |

## 分數校準指南

### 常見的評分偏誤

1. **過於寬鬆**：所有論文都給 6-7 分 -> 缺乏區別度
2. **過於嚴格**：除非完美否則不給高分 -> 打壓好的但不完美的工作
3. **錨定效應**：被第一印象錨定，後續分析無法改變判斷
4. **光環效應**：因為作者或機構有名而給高分
5. **新穎性偏見**：過度重視新穎性而忽略其他維度

### 校準建議

- 回想你最近讀過的同領域最好的論文，它應該是 8-9 分
- 回想你讀過的剛好可以接受的論文，它應該是 6-7 分
- 想像該會議的 oral paper，它們通常是 8-10 分
- 想像該會議的 poster paper，它們通常是 6-8 分
- 如果你猶豫不決，問自己：「這篇論文會讓會議變好還是變差？」

### 分數與建議的對應

| 分數範圍 | 建議 | 含義 |
|----------|------|------|
| 8-10 | Accept | 應該被接受，為會議增添價值 |
| 6-7 | Borderline / Weak Accept | 可以被接受，但有改進空間 |
| 5 | Borderline | 可接受可拒絕，取決於其他因素 |
| 3-4 | Reject / Weak Reject | 不應被接受，但可修改後重投 |
| 1-2 | Strong Reject | 明確不應被接受 |
