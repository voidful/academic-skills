# Output Template

Use this as the default structure. Adapt depth to corpus size, but preserve the section order.

**Language**: Output in Traditional Chinese (繁體中文) by default. Keep paper titles, venue names, method names, and technical terms in English. Switch to English if the user writes in English.

# Professor Snapshot
- identity and research positioning
- one-paragraph overall impression
- evidence coverage note

# Research Timeline
- key phases by year range
- major venue and theme changes
- notable collaboration or student patterns

# Research Taxonomy
## Core lines
## Secondary lines
## Application branches
## Shared methodological spine
## Recent pivots
## Likely expansion directions

# Paper-by-Paper Analysis
For each paper, include:
- metadata
- problem
- method
- novelty
- resources
- authorship analysis
- impact and signal
- writing quality
- confidence

# Advisor Assessment
## Research strength
## Mentoring strength
## Collaboration network
## Resource profile
## Workload and culture risk
## Teaching and communication signal

For each subsection, include:
- summary judgment
- supporting evidence
- counter-evidence or alternative explanations
- confidence

# Future Directions
- likely next directions
- where help is needed most
- durable versus opportunistic bets

# Fit Analysis for the Applicant
- strongest fit dimensions
- professor-student type matching
- main gaps
- probability of traction if contacting now
- best positioning strategy

# Warning Signs
- research direction mismatch indicators
- student development concerns
- collaboration dependency risks
- lab dynamics signals
- severity assessment for each (blocking / serious / minor)
- benign explanations where applicable

# Cold Email Templates
- student version
- collaborator version
- short relationship-building version

# Conversation and Interview Playbook
- likely questions
- what each question tests
- answer framework
- mistakes to avoid
- smart questions to ask back

# Final Verdict
- best-fit applicant type
- worst-fit applicant type
- biggest opportunity
- biggest risk
- next three actions
