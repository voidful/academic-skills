# 各會議格式規範

## NeurIPS (Neural Information Processing Systems)

- **頁數限制**: 正文 9 頁，參考文獻不計
- **格式**: 雙欄，`neurips_20XX.sty`
- **匿名審稿**: 是（雙盲）
- **附錄**: 允許，不計入頁數限制
- **程式碼**: 鼓勵提交，可附於 supplementary
- **審稿週期**: 約 3-4 個月
- **錄取率**: ~25%
- **重點審查**: 新穎性、理論嚴謹度、實驗完整性

## ICLR (International Conference on Learning Representations)

- **頁數限制**: 正文 8 頁 + 不限附錄
- **格式**: 雙欄，`iclr20XX_conference.sty`
- **匿名審稿**: 是（雙盲），OpenReview 公開審稿
- **特色**: 審稿意見公開可查
- **程式碼**: 強烈鼓勵
- **審稿週期**: 約 3 個月
- **錄取率**: ~30%
- **重點審查**: 清晰度、貢獻顯著性、可重現性

## ICML (International Conference on Machine Learning)

- **頁數限制**: 正文 8 頁，參考文獻不計
- **格式**: 雙欄，`icml20XX.sty`
- **匿名審稿**: 是（雙盲）
- **附錄**: 允許
- **審稿週期**: 約 3-4 個月
- **錄取率**: ~25%
- **重點審查**: 技術深度、創新性

## ACL / EMNLP / NAACL (NLP 系列)

- **頁數限制**: 長論文 8 頁，短論文 4 頁（+參考文獻）
- **格式**: 雙欄，ACL style
- **匿名審稿**: 是（雙盲）
- **ARR 系統**: ACL Rolling Review 統一審稿
- **錄取率**: ~22-25%
- **重點審查**: 語言學動機、baseline 完整性、error analysis

## CVPR / ECCV / ICCV (Computer Vision 系列)

- **頁數限制**: 正文 8 頁 + 參考文獻 (CVPR/ICCV)，14 頁 (ECCV)
- **格式**: 雙欄，各自 style file
- **匿名審稿**: 是（雙盲）
- **supplementary**: 允許影片與額外結果
- **錄取率**: ~25-30%
- **重點審查**: 視覺結果品質、與 SOTA 比較、ablation 完整性

## AAAI (Association for the Advancement of AI)

- **頁數限制**: 正文 7 頁 + 1 頁參考文獻
- **格式**: 雙欄，AAAI style
- **匿名審稿**: 是（雙盲）
- **錄取率**: ~20%

## 通用注意事項

### 投稿前檢查清單

- [ ] 頁數是否符合限制
- [ ] 是否使用正確的 style file 和年份版本
- [ ] 匿名化是否完整（姓名、機構、致謝、自引）
- [ ] 參考文獻格式是否正確
- [ ] 圖表是否清晰可讀（列印品質）
- [ ] Supplementary material 是否在大小限制內

### 常見格式錯誤

1. 使用舊版 style file
2. 正文超過頁數限制
3. 匿名化不完整（忘記移除致謝）
4. 圖片解析度太低
5. 參考文獻缺少年份或會議名稱
